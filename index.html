Access assets using the following URL pattern:

https://[your-domain]/assets/{folder}/{filename}

Examples:

  • /logos/chatgpt.png - Company logo
  • /screenshots/chatgpt-1.png - Application screenshot
  • /banners/chatgpt.png - Banner image
  • /categories/ai-tools.png - Category icon
